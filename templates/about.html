About CyberShield

CyberShield is a web-based cybersecurity project developed to identify suspicious and phishing URLs using rule-based detection techniques.

Project Objective

The objective of this project is to help users identify potentially dangerous websites before visiting them.

Technologies Used

  • Python
  • Flask
  • HTML5
  • CSS3
  • JavaScript

Detection Features

  • HTTPS Check
  • URL Length Check
  • IP Address Detection
  • Short URL Detection
  • Suspicious Keyword Detection
  • Risk Score Calculation
